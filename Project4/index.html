Kaylem Brown-Malone GetNewsOnTopics(string[] topics) : returns string[] yes Easy TRYIT This API can return up to 5 URLs for news stories related to given topics. Topics are provided in a “topics” query, comma-seperated. Uses the NewsAPI API, and the provided NewsAPI package.
Kaylem Brown-Malone GetAnnWindSpeeds(double latitude, double longitude) : returns double yes Easy TRYIT This API can return the Annual wind speed (in m/s at 10m) for any given latitude and longitude point. Uses the NASA POWER API, climatology W10MS endpoint.
Kaylem Brown-Malone VideoSearchService(string topic) : returns string no Easy TRYIT This API can return the top 5 Youtube videos for the given topic. Uses Google's API.
Kaylem Brown-Malone GenerateWorkoutService(string difficulty) : returns string no Easy TRYIT This API returns a bodyweight workout with the given difficulty. No external sources.
Kaylem Brown-Malone WeightLoggingService(int weight, string apikey) : returns boolean no Easy TRYIT This service is used to log a user's weight to their account. This test version checks for weight and apikey='test'. No external sources.
Sherwin Wang StockOpenPrice(string ticker, int month, int day, int year) : returns double yes Easy TRYIT This service returns the open price of a stock given a TICKER and date. Utilizes the polygon.io stock open price API.
Sherwin Wang FindNearestStore(int zipcode, string storeName) : returns string yes Easy TRYIT This service returns the nearest store given the zipcode and the store name. Utilizes google geocode and places APIs.
Sherwin Wang FindNearestGym(int zipcode) : returns string no Easy TRYIT This service returns a string of names and addresses of the top 5 closest gyms given the zip code. Utilizes google geocode and places APIs.
Sherwin Wang FindRecipes(string ingredients) : returns string no Easy TRYIT This service returns 5 names and images of recipes for the given ingredients. Ingredients are given in a list separated by commas. Utilizes the spoontacular Recipe - Food - Nutrition API.
